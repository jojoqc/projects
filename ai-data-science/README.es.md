# 🤖📊 Portafolio de Inteligencia Artificial, Machine Learning & Ciencia de Datos

Este repositorio documenta mi trabajo práctico y proyectos en **Inteligencia Artificial, IA Generativa, Retrieval-Augmented Generation (RAG) y Ciencia de Datos avanzada**.
